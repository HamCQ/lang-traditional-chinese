## Flarum 正體中文語言包 / Traditional Chinese Language Pack

### 需求 / Requirement
  - Flarum v1.0.0 or newer

### 安裝 / Installation
```
composer require pattinsonfuture/lang-traditional-chinese
composer require pattinsonfuture/lang-traditional-chinese
```

### 移除 / Remove
```
composer remove pattinsonfuture/lang-traditional-chinese
composer remove pattinsonfuture/lang-traditional-chinese
```

### 在地化 / Localization
  - 調整時間格式
  - 改進正體中文字型顯示
